# 4 Заключение — ML-составляющая

Настоящий файл содержит ML-составляющую заключения, организованную по задачам, поставленным во Введении (Тамразов.docx, разделы «Введение» — задачи 3, 4, 5; задачи 1–2 относятся к протоколу записи и предобработке сигналов и в данный файл не включены). Каждый вывод сопровождён ссылкой на конкретный результат подразделов 3.1–3.8.

## 4.1 Выводы по задачам ТЗ (ML-часть)

### 4.1.1 Задача 3: сформировать набор признаков для онлайн-предсказания

Зафиксирован информативный вклад каждой модальности по таргетам (подраздел 3.2, таблица 3.2): для LT2 — ВСР (субъект-взвешенная медианная MAE 1,90 мин, ρ = 0,91, R² = 0,82); для LT1 — комбинация ЭМГ+NIRS (MAE 2,11 мин, ρ = 0,74, R² = 0,52). Установлено, что добавление третьей модальности к ведущей не даёт значимого улучшения ни для LT2 (HRV vs EMG+NIRS+HRV, p = 0,966), ни для LT1 (EMG+NIRS vs EMG+NIRS+HRV, p = 0,442) — подраздел 3.2, таблица 3.3. Установлено также, что для нейросетевых архитектур абсолютные признаки эквивалентны их отсутствию в 86 % конфигураций (подраздел 3.4.1, таблица 3.5), тогда как для линейного семейства абсолютный признак `hrv_mean_rr_ms` несёт 44,6 % суммарного веса коэффициентов Ridge (подраздел 3.4.2, таблица 3.8) и его удаление снижает R² с 0,82 до 0,68.

### 4.1.2 Задача 4: разработать ML-модель; сопоставить линейный baseline и нейросетевые архитектуры

Построен и количественно сопоставлен зоопарк из десяти семейств (одно линейное и девять нейросетевых; раздел 2.8 методов). На LOSO-валидации с n = 18 испытуемыми установлено, что линейное семейство (Ridge / ElasticNet / HuberRegressor / GradientBoosting / SVR на ручных признаках) статистически значимо превосходит все девять нейросетевых архитектур (LSTM, Attention-LSTM, TCN, Monotonic TCN, четыре конфигурации вейвлет-семейства, ансамбль) — суммарно 90 значимых побед и 0 значимых поражений в 14 из 16 групп target × feature_set × variant по критерию Уилкоксона с поправкой Холма: в категории with_abs — 39 значимых побед в 6 из 8 групп (на модальности NIRS значимых различий не выявлено), в категории noabs — 51 значимая победа во всех 8 группах (подраздел 3.3, таблица 3.4). Постобработка фильтром Калмана (раздел 2.10) для финальных моделей систематического выигрыша не дала.

### 4.1.3 Задача 5: оценить точность LT1 и LT2; SHAP; сопоставить с литературой

Точность финальных моделей оценена в трёх дополняющих представлениях (раздел 2.9.1): субъект-взвешенная медианная MAE, raw window-weighted MAE, медиана по окнам. Полные значения по победителям приведены в подразделе 3.1 (таблица 3.1) и в дополнительной структуре ошибки в подразделе 3.5 (таблицы 3.9, 3.10). Установлено: для LT2 ошибка стабильна на всех бинах времени до порога (1,5–2,5 мин), для LT1 — стабильна на бинах ≤ 480 с (1,5–2,8 мин) и возрастает до 6,2–7,1 мин в бине > 480 с до порога (подраздел 3.5.2, таблица 3.10). Установлено также, что эта структура ошибки концентрируется в подгруппе тренированных испытуемых: критерий Манна–Уитни для LT1 на всех трёх топовых моделях даёт p = 0,0002…0,003 (подраздел 3.5.1, таблица 3.9), для LT2 — p = 0,35…0,68.

Анализ важности признаков методом SHAP (подраздел 3.7) выполнен для четырёх финальных моделей: для LT2-победителей — на Ridge с LinearExplainer, для LT1-победителей — на фактической модели SVR (C = 100, ε = 1,0; RBF) с PermutationExplainer. Для LT2 / HRV 75 % суммарной важности приходится на два признака — `hrv_mean_rr_ms` (43,5 %) и `hrv_dfa_alpha1` (31,4 %); для LT1 / EMG+NIRS доминирующего признака нет (ведущий — `trainred_hhb_std`, 5,2 %; суммарная доля топ-10 — 37,1 %), вклад распределён между EMG (54,0 % по группе), NIRS (40,3 %) и кинематикой (5,7 %). В мультимодальной модели LT1 / EMG+NIRS+HRV первым по важности оказывается `hrv_mean_rr_ms` (4,9 %), доля группы HRV — 11,6 %.

Сопоставление с литературой выполнено в подразделе 3.9 по шести направлениям: диссоциация модальностей по порогам (3.9.1), доминирование линейного семейства при малом n (3.9.2), зависимость роли абсолютных признаков от класса модели (3.9.3), эффект подгруппы тренированных (3.9.4), диффузность LT1 и концентрированность LT2 в SHAP (3.9.5), гетерогенность выборки (3.9.6).

### 4.1.4 Дополнительные результаты сверх задач ТЗ

Проверена гипотеза о двух специалистах для LT1 (раздел 2.8.9, подраздел 3.6). Установлено: симметричный вариант — раздельные модели для тренированных и нетренированных — не превосходит единую обобщённую модель ни при идеальной маршрутизации (p = 0,246), ни при реальной (p = 0,899). Асимметричный вариант — специалист для нетренированных + generalist для тренированных — значимо превосходит единую модель при идеальной маршрутизации (p = 0,034), при реальной маршрутизации даёт тренд к значимости (p = 0,074, MAE_median 3,15 → 2,18 мин). Точность маршрутизатора на базальных признаках — 66,7 %, что совпадает с базовой частотой большего класса.

Дополнительно построены LOSO split-conformal интервалы для четырёх финальных моделей в обеих категориях признаков — итого 16 строк (раздел 2.11.4, подраздел 3.8, таблица 3.17). Эмпирическое покрытие отклоняется от целевого в пределах 0,2–2,2 п.п. Полуширина 80 %-интервалов с абсолютными признаками: LT2 / HRV — ±3,49 мин, LT1 / EMG+NIRS — ±6,99 мин. Удаление абсолютных признаков расширяет интервалы LT2 (для HRV — до ±4,87 мин, +1,38 мин) и не меняет интервалы LT1 (для EMG+NIRS — ±6,59 мин, даже сужение на 0,40 мин).

## 4.2 Научная новизна

Каждый пункт новизны сопровождён ссылкой на конкретный результат настоящей работы; формулировки ограничены тем, что подкреплено собственным результатом.

1. Количественно показана зависимость минимально достаточного набора сенсоров от типа порога: для LT2 одной модальности ВСР достаточно (MAE 1,90 мин; добавление ЭМГ и NIRS не значимо, p = 0,966); для LT1 ведущей является комбинация ЭМГ+NIRS (MAE 2,11 мин; добавление ВСР не значимо, p = 0,442). Подраздел 3.2.
2. Установлено, что выбор «абсолютные признаки vs тренды» зависит от класса модели: для нейросетевых архитектур — эквивалентность в 86 % конфигураций, для линейного семейства — статистически значимое преимущество with_abs на LT2 для модальности HRV (Δ_median MAE = +0,54 мин; p = 0,006) и для мультимодальной EMG+NIRS+HRV (Δ = +0,53; p = 0,034); ведущий абсолютный признак — `hrv_mean_rr_ms` (доля 44,6 % коэффициентов Ridge HRV / LT2). Для LT1 ablation линейного семейства даёт эквивалентность во всех модальностях. Подраздел 3.4.
3. Получено эмпирическое подтверждение, что при n = 18 регуляризованная линейная регрессия на ручных признаках статистически значимо превосходит все девять протестированных нейросетевых архитектур (90 значимых побед, 0 поражений в 14 из 16 групп target × feature_set × variant; paired Wilcoxon + Holm). Подраздел 3.3.
4. Описан и количественно охарактеризован эффект подгруппы тренированных для предсказания LT1: per-subject MAE значимо выше на тренированных (p = 0,0002…0,003); 79 % окон с |err| > 7 мин приходятся на эту подгруппу. Эффект отсутствует для LT2 (p = 0,35…0,68). Подразделы 3.5, 3.8.
5. Показано, что для LT1 при n = 18 симметричное разделение модели на двух специалистов не работает, тогда как асимметричный вариант (исключение нетипичной подгруппы из обучающей выборки модели большинства) даёт значимый выигрыш при идеальной маршрутизации (p = 0,034) и тренд при реальной (p = 0,074). Подраздел 3.6.

## 4.3 Практическая значимость

1. Минимальная конфигурация датчиков по таргетам: для онлайн-предсказания LT2 достаточно носимого кардиомонитора с регистрацией RR-интервалов; для LT1 — комбинации поверхностной ЭМГ и NIRS на работающей мышце. Подраздел 3.2.
2. Для нейросетевых архитектур индивидуальная калибровка абсолютных норм физиологических сигналов не требуется; для линейного семейства калибровка `hrv_mean_rr_ms` (средний RR на калибровочной ступени) необходима. Подраздел 3.4.
3. Для LT2 предсказательные интервалы практически применимы: полуширина ±3,49 мин при 80 %-покрытии (LOSO split-conformal, n = 18). Для LT1 интервалы шире — ±6,99 мин при 80 %, ±9,05 мин при 90 % — и в текущей версии не достаточны для оперативного использования. Подраздел 3.8.

## 4.4 Ограничения

1. Объём выборки n = 18 испытуемых (записано 19, одна запись исключена из-за низкого качества сигнала). Все результаты следует рассматривать в этом масштабе; для отдельных нейросетевых архитектур по таргету LT2 число субъектов в LOSO составило 16–17 из-за внутренних `dropna` (подраздел 3.1, таблица 3.1).
2. Выборка гетерогенная и отклонилась от исходно заявленной целевой популяции — триатлонистов (раздел 1.7 обзора литературы). В связи с трудностями набора целевой группы она была расширена до лиц с различным уровнем физической подготовки. Это ограничение работы; вместе с тем гетерогенность обернулась методологическим преимуществом — позволила количественно обнаружить эффект подгруппы тренированных (подразделы 3.5–3.6), на однородной выборке невидимый. Гетерогенность не была изначальным замыслом исследования.
3. Подгруппа тренированных (n = 6) численно мала. Результаты по системе двух специалистов (подраздел 3.6) являются эксплоративными.
4. Маршрутизатор в системе двух специалистов достигает точности 66,7 % — на уровне базовой частоты большего класса. Это главное узкое место системы; устранение требует расширения выборки и дополнительных базальных признаков.
5. Для LT1 LOSO split-conformal интервалы имеют полуширину ±7–9 мин (подраздел 3.8). При текущем объёме данных интервал недостаточно узок для оперативного использования по аэробному порогу.
6. Расхождение между тремя представлениями MAE сохраняется и должно учитываться при цитировании числовых значений: для LT1 / EMG+NIRS субъект-взвешенная медианная MAE равна 2,11 мин, медиана |err| по окнам — 3,33 мин, raw window-weighted MAE — 4,25 мин (подраздел 3.8, ссылка на step_06).
## 4.5 Направления дальнейших исследований

1. Расширение выборки, в первую очередь — подгруппы тренированных, для проверки гипотезы о двух симметричных специалистах и устранения главного ограничения системы двух специалистов (подраздел 3.6).
2. Поиск концентрированного физиологического маркера LT1 — агрегированного признака, способного синтезировать вклад множества слабых сигналов в одну величину. Обоснование — диффузный SHAP-профиль LT1 (подраздел 3.7.3, 3.9.5).
3. Перевод маршрутизатора подгрупп на более информативные базальные признаки; в текущей версии маршрутизатор работает на уровне базовой частоты классов (подраздел 3.6).
4. Расширение conformal-калибровки от глобальной к стратифицированной по подгруппам тренированности — для отражения структурной неоднородности per-subject ошибки LT1 в виде различий полуширин (подраздел 3.8).
5. Расширение SHAP-интерпретации на нейросетевые архитектуры (DeepExplainer / Integrated Gradients для TCN, Attention-LSTM, вейвлет-семейства). Это позволит количественно сопоставить, какие признаки используют классы моделей, для которых не выявлено превосходства над линейным семейством (раздел 3.3), и проверить гипотезу о структурно ином использовании временного контекста (раздел 3.9.3).
