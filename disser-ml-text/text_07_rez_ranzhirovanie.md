## 3.1 Сводный рейтинг моделей

В данном подразделе приведены количественные результаты широкого перебора, описанного в разделе 2.8.4: единое линейное семейство и девять нейросетевых архитектур, обученные раздельно для двух таргетов (LT1, LT2) на пяти модальностях и в двух вариантах (with_abs / noabs). Все метрики получены в рамках LOSO-валидации на n = 18 испытуемых (раздел 2.8.5) и приведены к субъект-взвешенной медианной MAE в минутах (раздел 2.9.1). В работе используются три дополняющих представления MAE: субъект-взвешенная медианная MAE (median per-subject MAE; основная метрика рейтингования), медиана |err| по окнам (window-median) и raw window-weighted MAE (среднее по всем окнам без межсубъектного выравнивания). Коэффициент детерминации R² и ранговая корреляция Спирмена ρ вычислены согласно разделам 2.9.2 и 2.9.3 соответственно.

Главная сводка результатов представлена в таблице 3.1. Для каждого таргета в неё включены три позиции: победитель в абсолютных метриках, мультимодальный аналог (на полном наборе EMG+NIRS+HRV) и лучшая нейросетевая конфигурация.

**Таблица 3.1.** Финалисты по каждому таргету (LOSO, n = 18; для отдельных нейросетевых конфигураций n = 17, см. раздел 2.8.5).

В обоих случаях победителем является конфигурация линейного семейства (раздел 2.8.6); статистическая проверка превосходства линейного семейства над нейросетевыми архитектурами вынесена в подраздел 3.3.

## 3.2 Диссоциация модальностей по порогам

Для оценки информативного вклада каждой модальности был построен лучший представитель на каждом наборе признаков (таблица 3.2). Сравнение проведено внутри одного класса модели — линейного семейства, что устраняет вклад различий в архитектуре и оставляет в качестве источника вариации только состав признаков.

**Таблица 3.2.** Лучшая модель по каждой модальности (линейное семейство, with_abs; субъект-взвешенная медианная MAE).

| NIRS | 3,62 | 0,72 | 3,45 | 0,62 |
| HRV | 3,51 | 0,65 | **1,90** | **0,91** |
| EMG+NIRS | **2,11** | **0,74** | 2,90 | 0,70 |
| EMG+NIRS+HRV | 2,23 | 0,79 | 1,91 | 0,91 |

Источник: [results/final/dissertation_tables/table_by_modality.csv](results/final/dissertation_tables/table_by_modality.csv).

В таблице 3.2 выявлена ключевая закономерность: оптимальная модальность зависит от таргета. Для LT2 лучшие показатели даёт ВСР (1,90 мин); комбинирование её с ЭМГ и NIRS не улучшает результат (1,91 мин). Для LT1, напротив, ведущей является комбинация ЭМГ+NIRS (2,11 мин); добавление ВСР здесь незначительно ухудшает медиану (2,23 мин) при росте ρ (0,79 против 0,74). Сама по себе модальность ВСР для LT1 даёт MAE 3,51 мин — в 1,7 раза хуже, чем ЭМГ+NIRS.

Для верификации этого наблюдения проведены кросс-модальные парные сравнения по критерию Уилкоксона с поправкой Холма (раздел 2.11.1). Полная сводка приведена в таблице 3.3.

Полученные результаты количественно фиксируют:
1. В категории **with_abs** для LT2 одной модальности ВСР статистически достаточно (HRV vs EMG+NIRS+HRV p = 0,966), и ВСР значимо превосходит EMG+NIRS (p = 0,012); для LT1 достаточно ЭМГ+NIRS (добавление ВСР p = 0,442).
2. В категории **noabs** для LT2 все три модальности (HRV, EMG+NIRS, EMG+NIRS+HRV) статистически эквивалентны (p ≥ 0,32) — преимущество HRV из with_abs опирается именно на абсолютный признак `hrv_mean_rr_ms`; для LT1 наблюдается обратная картина — добавление ВСР даёт значимое улучшение (EMG+NIRS+HRV vs EMG+NIRS, p = 0,038).

Таким образом, минимально достаточный набор сенсоров не универсален, а зависит как от целевого порога, так и от того, доступны ли индивидуальные базальные уровни (которые требуются для абсолютных признаков). Физиологическая интерпретация этой диссоциации вынесена в подраздел 3.9.
