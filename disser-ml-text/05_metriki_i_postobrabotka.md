# 2.8.5 Метрики качества и постобработка предсказаний

## О чём писать
- **Метрики**:
  - MAE (минуты) — основная, интерпретируемая.
  - Различать три варианта MAE: `loso_mae_median` (медиана по субъектам),
    median по окнам, raw MAE (среднее по окнам). Объяснить, почему по субъектам
    оптимистичнее, и какую цитируем где.
  - R² — доля объяснённой дисперсии.
  - ρ (Спирмен) — ранговое качество (часто важнее MAE для задачи «когда наступит»).
- **Постобработка — фильтр Калмана**: предсказания по окнам сглаживаются;
  σ_obs ∈ {30, 50, 75, 150}. Метрика `kalman_gain` = выигрыш от сглаживания.
- **Honest baselines** (для контекста): MeanPredictor и FirstWindowPredictor —
  показать, что модель бьёт тривиальные предсказания (−58…−59% MAE).

## Откуда брать
- `results/final/01_ranking_wide.csv` — все метрики.
- Honest baselines — `results/v0011/honest_baselines.csv`.
- Фильтр Калмана — реализация в `scripts/v0011_modality_ablation.py`.

## Как собирать
1–1,5 страницы. Формулы MAE/R²/ρ кратко. Обязательно — абзац про «три MAE»,
иначе в результатах будет путаница в числах. Калман — пара предложений + зачем.
