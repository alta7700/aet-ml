# 4 Заключение — ML-вклад

## О чём писать
Заключение строится **по задачам ТЗ как по чеклисту** (вывод на каждую задачу).
Ниже — ML-составляющие, которые войдут в общее заключение работы.

### Выводы по задачам (ML-часть)
- По задаче «построить модель предсказания порога»: достигнута MAE 1,86 мин (LT2)
  и 2,11 мин (LT1) при LOSO-валидации, n=18; модель значимо превосходит
  тривиальные baseline.
- По задаче «сравнить подходы ML»: регуляризованная линейная регрессия значимо
  превосходит 9 нейросетевых архитектур (Уилкоксон + Холм).
- По задаче «оценить вклад модальностей»: установлена диссоциация — LT1
  предсказывается по ЭМГ+NIRS, LT2 — по ВСР; мультимодальность не даёт значимого
  выигрыша.
- По задаче «персонализация»: гипотеза двух экспертов подтверждена в асимметричном
  варианте (специалист для нетренированных).

### Научная новизна (со ссылкой на результаты)
- Впервые показана зависимость оптимального набора сенсоров от типа порога.
- Показано, что выбор «абсолютные признаки vs тренды» определяется классом модели.

### Практическая значимость
- Минимальная конфигурация датчиков: для LT2 достаточно кардиомонитора (ВСР),
  для LT1 — ЭМГ+NIRS.
- Не требуется индивидуальная калибровка абсолютных норм.
- Для LT2 — практически применимые интервалы неопределённости (±3,5 мин, 80%).

### Ограничения
- n=18 (записано 19, одна запись исключена из-за низкого качества сигнала).
- Выборка гетерогенная и отклонилась от исходно заявленной целевой популяции
  (триатлонисты): из-за трудностей набора целевой группы она была расширена до
  лиц с различным уровнем подготовки. Подавать честно как ограничение — но с
  оговоркой, что гетерогенность обернулась методологическим преимуществом
  (проверка обобщаемости, обнаружение эффекта подгруппы тренированных).
  НЕ выдавать гетерогенность за изначальный замысел.
- Подгруппа тренированных n=6 — результаты по «двум экспертам» эксплоративные.
- Маршрутизатор достигает точности лишь на уровне base rate.
- Для LT1 conformal-интервалы слишком широки для практики.

### Направления дальнейших исследований
- Расширение выборки, особенно тренированных.
- Улучшение разметки LT1 (диффузность сигнала).
- Маршрутизатор на более информативных признаках калибровки.
