# 2.9 Статистическая обработка данных

В настоящем разделе описываются статистические методы, применявшиеся для
сравнения моделей и проверки исследовательских гипотез. Конкретные результаты
тестов (p-значения, размеры эффекта, выводы о превосходстве конкретных
конфигураций) приводятся в главе 3.

## Парные сравнения моделей: критерий Уилкоксона с поправкой Холма

Для сравнения двух произвольных конфигураций моделей (по версии, набору признаков
или варианту) применялся **парный критерий знаковых рангов Уилкоксона** (двусторонний). Тест
строился на векторах per-subject MAE: для каждого испытуемого вычислялась MAE
на его тестовом фолде, и сравнивались пары значений MAE одного и того же
испытуемого под двумя конфигурациями. При сравнении моделей с разным числом
субъектов (n = 16, 17 или 18) в анализ включались только испытуемые,
присутствующие в обоих тестовых фолдах. Пары с числом совпадающих субъектов
менее 12 отмечались как недостаточно мощные и в финальные выводы не включались.

При проведении множественных попарных сравнений внутри одной группы
(target × feature\_set × variant) применялась **поправка Холма (Holm–Bonferroni)**
на контроль familywise error rate. Порог значимости: **α = 0,05**.

Вычисления выполнялись средствами библиотеки scipy 1.17.1; поправка Холма —
методом Holm–Bonferroni.

## Сравнение независимых групп: критерий Манна–Уитни

Для проверки различий между тренированными и нетренированными испытуемыми
применялся **критерий Манна–Уитни** (Mann–Whitney U-test, двусторонний), поскольку
эти подгруппы являются независимыми выборками. Сравнивались распределения per-subject MAE внутри двух подгрупп: 6 тренированных
(время до LT2 более 13 мин) и 12 нетренированных. Вычисления — средствами
библиотеки scipy 1.17.1.

## Исключение абсолютных значений: критерий эквивалентности

Для оценки практической значимости вариантов with\_abs и noabs использовалась
операциональная **предметная граница эквивалентности**:

- Различие признаётся **несущественным** («эквивалентно»), если одновременно
  выполняются: $|\Delta\,\mathrm{median}| < 0{,}3$ мин И $p \geq 0{,}05$
  (парный Уилкоксон на per-subject MAE);
- При нарушении хотя бы одного условия выносится вердикт «лучше with\_abs»
  или «лучше noabs» в зависимости от знака $\Delta$.

Порог 0,3 мин (~18 с) отражает минимальное клинически различимое улучшение:
отклонение меньше этого значения не изменяет практических рекомендаций по
выбору набора сенсоров. Поправка Холма в данном аблейшн-анализе не применялась —
он носит поисковый (exploratory), а не подтверждающий (confirmatory) характер.
Анализ проводился по 9 нейросетевым версиям (v0101–v0107) на пересечении
субъектов, присутствующих в обоих вариантах. Линейная модель v0011 в данный
анализ не включалась, поскольку для неё вариант noabs не имеет сохранённых
предсказаний (причины описаны в разделе 2.8.3).

## Конформное предсказание (conformal prediction)

Для построения предсказательных интервалов с гарантированным покрытием применялся
метод **LOSO split-conformal prediction**. Метод основан на принципе обменности
(exchangeability) остатков модели: при условии, что тестовый объект обменим с
обучающими, интервал с целевым покрытием $1 - \alpha$ выполняется на конечных
выборках без параметрических допущений о распределении ошибок.

В реализации LOSO split-conformal:

1. На каждой итерации LOSO обученная модель применялась к тестовому субъекту;
   его предсказания формировали «тестовый» набор.
2. Остатки модели на оставшихся 17 субъектах (обучающий фолд) использовались
   как калибровочный набор для вычисления квантиля конформных оценок
   $q_{1-\alpha}$.
3. Предсказательный интервал для окна $t$ тестового субъекта:
   $$
   \hat{y}_t \pm q_{1-\alpha}^{\text{cal}}.
   $$

Тестировались два уровня покрытия: **90%** ($\alpha = 0{,}1$) и **80%**
($\alpha = 0{,}2$). Уровень покрытия задаёт, какую долю тестовых окон интервал
должен накрывать: при 90% истинное значение должно попадать в интервал не менее
чем в 9 случаях из 10. Более высокий уровень даёт более широкий интервал.

Качество интервалов оценивалось по двум характеристикам:
- **эмпирическое покрытие** — фактическая доля тестовых окон, в которые попало
  истинное значение (сравнивается с целевым уровнем);
- **полуширина** (в минутах) — величина $q_{1-\alpha}$, то есть отклонение
  «±» от точечного предсказания.

Анализ проводился для финальных моделей-победителей (раздел 2.8.3): линейная
регрессия на HRV и на EMG+NIRS+HRV для LT2; линейная регрессия на EMG+NIRS и
на EMG+NIRS+HRV для LT1. Покрытие оценивалось раздельно для тренированных и
нетренированных субъектов.

## Интерпретация модели: анализ важности признаков (SHAP)

Чтобы понять, какие физиологические сигналы вносят наибольший вклад в
предсказание времени до порога, применялся метод SHAP (SHapley Additive
exPlanations). Для каждого окна SHAP разбивает предсказание модели на вклады
отдельных признаков: положительный вклад означает, что признак сдвигает
предсказание вверх (дальше от порога), отрицательный — вниз (ближе к порогу).

Для линейных моделей семейства v0011 значения SHAP вычислялись аналитически
с помощью линейного SHAP-explainer (библиотека shap 0.51.0) — без приближений.

Важность каждого признака оценивалась как среднее абсолютное значение SHAP
по всем тестовым окнам всех субъектов $\bar{|\phi_j|}$. Дополнительно признаки
объединялись в группы по модальности (EMG, NIRS, HRV, кинематика,
взаимодействия) — это позволяет оценить относительный вклад каждого источника
сигнала в целом, а не отдельных признаков.

Стандартный набор визуализаций включал: beeswarm-диаграмму (распределение
значений SHAP по признакам), bar-диаграмму средних абсолютных вкладов признаков
и waterfall-диаграмму для отдельных субъектов.
