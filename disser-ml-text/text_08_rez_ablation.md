## 3.4 Абсолютные против трендовых признаков

В рамках широкого перебора каждая модель обучалась в двух вариантах состава признаков — with_abs и noabs (раздел 2.7.6; перечень удаляемых в noabs признаков — там же). В данном подразделе количественно сопоставлены результаты обоих вариантов и установлено, в какой степени абсолютные признаки необходимы для предсказания. Сопоставление выполнено раздельно для нейросетевых архитектур и для линейного семейства, так как механизмы использования абсолютных уровней этих двух классов моделей различны (раздел 2.8.6 для линейной модели, 2.8.7 для нейросетевых).

### 3.4.1 Нейросетевые архитектуры

Для каждой из 72 нейросетевых конфигураций (9 архитектур × 4 модальности × 2 таргета) проведено парное сравнение with_abs и noabs по критерию Уилкоксона со знаковыми рангами на per-subject MAE (раздел 2.11.1) с применением операциональной границы эквивалентности (раздел 2.11.3). ### 3.4.3 Сводный вердикт

Установлено, что вердикт «нужны ли абсолютные признаки» зависит от класса модели:
- для нейросетевых архитектур абсолютные признаки не нужны: 86 % конфигураций эквивалентны вариантам noabs, а среди значимо различающихся преобладают конфигурации в пользу noabs;
- для линейного семейства абсолютный признак `hrv_mean_rr_ms` критичен: его удаление снижает R² на 0,14 (с 0,82 до 0,68).

Механизм этого расхождения связан с тем, как каждый из классов моделей обрабатывает временной контекст: линейное семейство в варианте v0011 предсказывает по отдельному окну (раздел 2.8.6) и не имеет внутренней рекуррентной памяти, поэтому абсолютный уровень `hrv_mean_rr_ms` служит для него межсубъектным «якорем» позиции на шкале нагрузки; нейросетевые архитектуры (раздел 2.8.7) видят последовательность окон и реконструируют положение на шкале нагрузки из временного контекста, поэтому абсолютный признак становится избыточным. Содержательная интерпретация этого расхождения вынесена в подраздел 3.9.
