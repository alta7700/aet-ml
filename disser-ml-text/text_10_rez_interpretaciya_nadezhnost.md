## 3.7 Интерпретация модели: анализ важности признаков (SHAP)

Для интерпретации линейного семейства на четырёх ключевых конфигурациях подразделов 3.1–3.2 — LT2 / HRV, LT2 / EMG+NIRS+HRV, LT1 / EMG+NIRS, LT1 / EMG+NIRS+HRV — выполнен анализ важности признаков методом SHAP (раздел 2.11.5). В качестве унифицированной модели интерпретации на всех четырёх конфигурациях использовалась Ridge с α = 1000 (LinearExplainer); унификация устраняет различия в инструменте интерпретации между конфигурациями и обеспечивает единый формат SHAP-значений. Соответствие унифицированной модели фактическому победителю линейного зоопарка различается по конфигурациям: для LT2 / HRV победитель — Ridge (α = 1000), полное совпадение; для LT2 / EMG+NIRS+HRV — ElasticNet (α = 1,0; l1 = 0,9), приближение в рамках регуляризованной линейной регрессии; для LT1 / EMG+NIRS и LT1 / EMG+NIRS+HRV — SVR с радиальным базисным ядром (C = 100, ε = 1,0), то есть модель с нелинейностью. Для двух LT1-конфигураций SHAP-значения на Ridge следует рассматривать как линейную проекцию вкладов признаков, а не как точную декомпозицию предсказания SVR; настоящее ограничение зафиксировано в подразделе 4.4 (Ограничения). Мера важности признака и групповая агрегация по модальности — по разделу 2.11.5; состав групп (EMG, NIRS, HRV, кинематика, признаки взаимодействия) — по разделу 2.7.
