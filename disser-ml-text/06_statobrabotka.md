# 2.9 Статистическая обработка данных

## О чём писать
Здесь описываются статистические **методы** (применение — в результатах):

- **Парные сравнения моделей** — критерий Уилкоксона (paired, signed-rank) на
  per-subject MAE; пересечение субъектов при разном n. Поправка Холма на
  множественные сравнения внутри группы. Порог α = 0,05.
- **Сравнение групп** (тренированные/нетренированные) — критерий Манна–Уитни
  (независимые выборки).
- **Conformal prediction** — LOSO split-conformal: интервалы предсказания с
  гарантией покрытия; α ∈ {0,1; 0,2}. Метод, не результат.
- **SHAP** — анализ важности признаков; для линейных моделей — LinearExplainer.
  Агрегирование вкладов по группам признаков.
- Порог «эквивалентности» для ablation: |Δ медиан| < 0,3 мин ∧ p ≥ 0,05.

## Откуда брать
- Реализации: `scripts/final_pairwise_wilcoxon.py`, `final_abs_vs_noabs.py`,

## Как собирать
1,5–2 страницы. По каждому методу: что это, на что применяется, какие пороги.
Не приводить результаты — только аппарат. Указать ПО и версии (требование стандарта).
