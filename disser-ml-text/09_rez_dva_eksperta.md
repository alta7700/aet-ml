# 3.x Результаты: тренированные vs нетренированные («два эксперта»)

## О чём писать
- **Эффект подгруппы**: для LT1 тренированные (time_to_lt2 > 13 мин, 6 чел.)
  предсказываются значимо хуже нетренированных (Манн–Уитни p=0,0002…0,003).
  Для LT2 эффекта нет (p=0,35…0,68).
- **Хвост ошибки структурирован**: 79% окон с ошибкой >7 мин — тренированные.
  mean MAE: тренированные 5,8 мин vs нетренированные 3,0 мин.
- **Время-разрешённый анализ**: для LT1 ошибка приемлема за 4 мин до порога, но
  6–7 мин при >8 мин до порога (это те же тренированные с длинными тестами).
  Для LT2 — стабильна на любом расстоянии до порога.
- **Гипотеза «два эксперта»**:
  - симметричный вариант (отдельная модель для каждой группы) **отвергнут** —
    специалист на n=6 недообучен, хуже generalist на своей же группе;
  - асимметричный вариант (специалист для нетренированных + generalist для
    тренированных) **подтверждён**: oracle p=0,034, реальный маршрутизатор
    p=0,074 (тренд), медианная MAE 3,15→2,18 мин.
- **Маршрутизатор**: логрег на базальных признаках (ЧСС@30Вт, % жира, возраст),
  точность 66,7% — на уровне base rate, главное ограничение системы.

## Как собирать
3–4 страницы — самый «сюжетный» результат. Логика: обнаружили эффект подгруппы →
подтвердили тремя методами (per-subject, time-resolved, conformal) → попытались
решить «двумя экспертами» → симметрично не вышло, асимметрично вышло частично.
Честно подать n=6 как ограничение. Это эксплоративный результат — так и назвать.
