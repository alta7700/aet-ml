## 2.9 Метрики качества

### 2.9.1 Три варианта МАЕ

Центральной метрикой в работе служит средняя абсолютная ошибка (MAE, mean absolute error). Поскольку при LOSO-валидации на 18 субъектах отдельные субъекты вносят в итоговую статистику неодинаковое число окон, различаются три связанных, но различных показателя:

- Среднее MAE по окнам — среднеарифметическое абсолютных ошибок по всем тестовым окнам всех субъектов. Субъекты с большим числом окон имеют больший вес.

- Медиана по окнам — медиана абсолютных ошибок по всем тестовым окнам объединённо (каждое окно имеет вес 1). Устойчива к выбросам; применяется при анализе распределения ошибки.

- Субъект-взвешенная медиана MAE — для каждого из 18 испытуемых вычислялась MAE по его тестовым окнам; итоговый показатель — медиана этих 18 значений. Каждый испытуемый имеет вес 1 вне зависимости от числа его окон. Это основная метрика в сводных таблицах настоящей работы.

Единица измерения для всех трёх вариантов — минуты (целевая переменная конвертируется из секунд при подсчёте MAE делением на 60).

### 2.9.3 Ранговая корреляция Спирмена (ρ)

Коэффициент ранговой корреляции Спирмена вычислялся между предсказанными и истинными значениями по всем тестовым окнам. Этот показатель устойчив к монотонным нелинейным искажениям шкалы и особенно информативен для задачи «когда наступит порог»: высокий ρ означает, что модель правильно ранжирует окна по близости к порогу даже при систематическом смещении абсолютных значений.

### 2.9.4 Базовые предсказатели

Для оценки нижней границы сложности задачи использовались два тривиальных предсказателя:

- MeanPredictor — предсказывает константу, равную среднему значению целевой переменной по обучающему множеству фолда; не использует ни физиологические сигналы, ни временну́ю структуру теста. Служит абсолютной нижней границей: модель, не превосходящая MeanPredictor, не извлекает никакой полезной информации.

- FirstWindowPredictor — предсказывает для каждого окна значение целевой переменной, зафиксированное в первом окне сессии данного испытуемого. Поскольку время до порога монотонно убывает в ходе теста, первое окно несёт информацию о начальной удалённости от порога даже без обращения к сигналам. Этот предсказатель проверяет более строгое условие: превзойти его означает, что физиологические сигналы добавляют информацию сверх одной лишь временно́й структуры нагрузочного теста.

Обе базовые линии воспроизводились в рамках LOSO совместно с основным экспериментом. Превышение точности модели над соответствующей базовой линией количественно характеризует вклад физиологических сигналов в предсказание.
