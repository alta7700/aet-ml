# Шаг 02a. Распределение per-subject ошибок у топ-моделей

## Цель
Понять, **как распределена ошибка по субъектам** у моделей-победителей шага 02. Это нужно:
- Для отбора кандидатов на «два эксперта» (если 6 тренированных лежат отдельно от 12 нетренированных по MAE — гипотеза подтверждается ещё до переобучения).
- Для текста диссертации: «модель работает однородно / есть pattern на тренированных».
- Для отчёта об ограничениях: какие субъекты «трудные».

## Входы
- `results/<version>/per_subject_full.csv` (10 файлов из шага 02-доп2).
- `dataset/session_params.parquet` — `hrv_hr_baseline_bpm` (из шага 01).
- `dataset/subjects.parquet` — антропометрия.
- Времена `time_to_lt1`/`time_to_lt2` из `targets.parquet` + `windows.parquet` (как считалось в проверке шага 01).

## Топ-модели для анализа
Из шага 02 — **по 3 модели на target**:

**lt2 (with_abs)**:
1. v0011 HRV Ridge(α=1000) — MAE_med=1.86
2. v0011 EMG+NIRS+HRV EN — MAE_med=1.89
3. v0107 ensemble EMG+NIRS+HRV — MAE_med=2.38

**lt1 (with_abs)**:
1. v0011 EMG+NIRS — MAE_med=2.11
2. v0011 EMG+NIRS+HRV — MAE_med=2.23
3. v0107 ensemble EMG+NIRS+HRV (noabs) — MAE_med=2.71

## Что считаем
Для каждой топ-модели:
1. **Гистограмма MAE по субъектам** + boxplot.
2. **Scatter `MAE_subject × HR_baseline_30W`**: проверка «нетренированные предсказываются хуже».
3. **Scatter `MAE_subject × time_to_lt2`**: проверка «короткие тесты сложнее».
4. **Scatter `MAE_subject × age`** и `× bmi` — антропометрические зависимости.
5. **Группы «trained / untrained» (по time_to_lt2 > 13 мин)**: средняя MAE в каждой группе, paired Wilcoxon между ними (если разница значима — обоснование шага 05).
6. **Кросс-target корреляция**: для каждого субъекта `MAE_lt1` vs `MAE_lt2` — есть ли «трудные везде».

**Вывод**:

- Для **lt1** тренированные предсказываются **в 2.5–3 раза хуже** нетренированных. Эффект очень значим (p < 0.005 на всех трёх топ-моделях).

6 из 8 худших — тренированные (5 из 6 топ-трудных). У S009 R² положительный, но MAE 6.8 мин — модель попадает в правильный диапазон, но грубо. У S003 R²<0 — модель прогнозирует хуже, чем медиана таргета.

Демография «трудных» тренированных: возраст 33–43 года; HR_baseline 66–96 bpm (низкая, как и положено тренированным); time_to_lt2 14.8–24 мин.

### Кросс-target корреляция MAE

`MAE_lt1 × MAE_lt2` (по 18 субъектам, v0011 EMG+NIRS+HRV): **ρ = +0.20, p = 0.43**.

Связи нет — это два разных мира предсказуемости. Субъект может быть «легко предсказуем» для LT2 и при этом «трудный» для LT1. Это даёт право в тексте рассматривать LT1 и LT2 как отдельные подзадачи.

### Готовность к шагу 05

Гипотеза «два эксперта» **доказана предварительно для lt1** и **отвергнута для lt2** (на этой выборке). В шаге 05 строим маршрутизатор **только для lt1**: специалисты trained / untrained + gate на основе baseline-признаков (HR@30W + антропометрия).

Ожидаемый выигрыш: на lt1 untrained модель уже работает с MAE 1.66 мин. Если добавить специалиста для trained с собственной MAE ~2.5 мин (за счёт обучения на их же 6 субъектах) → общая MAE_median может упасть с 2.23 до ~1.9 мин (медиана сместится в сторону untrained).
