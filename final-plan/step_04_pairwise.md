# Шаг 03. Парные сравнения моделей-кандидатов

## Цель
Получить право писать в тексте «модель X статистически значимо лучше Y», а не «MAE на 0.1 мин ниже».

## Входы
- Список кандидатов из `results/final/05_candidates_intersection.csv` (шаг 01)
- `results/<version>/per_subject.csv` всех версий из списка

## Что считаем
Для фиксированного `(target, feature_set, variant)`:
1. Берём список моделей-кандидатов в этой группе.
2. Для каждой пары моделей считаем **paired Wilcoxon** на per-subject MAE (только субъекты, присутствующие в обоих файлах).
3. Применяем **поправку Holm** на множественное сравнение внутри группы.
4. Сохраняем: модели A/B, n_paired, медиана разницы, p, p_holm, вердикт.

## Артефакты
- `scripts/final_pairwise_wilcoxon.py`
- `results/final/07_pairwise_wilcoxon.csv`
- `results/final/plots/pairwise_heatmap_<target>_<fset>.png` — матрица p_holm для каждой группы кандидатов
- Подпись в `results/final/README.md`

## Ограничения
- `n_paired` обязан быть ≥12, иначе строку помечаем `underpowered=True` и в текст не выносим.
- Только two-sided тест (мы не знаем заранее, кто лучше).

## Критерий завершённости
- Для каждой целевой группы есть либо победитель (p_holm<0.05), либо явная пометка «различия незначимы».
- В тексте диссертации можно ссылаться на конкретные строки CSV.

### Результаты

**Значимых различий: 168/648 пар** (p_holm < 0.05).

**v0011 (linear) — статистически доказанное доминирование** в категории `with_abs`:

| группа | net (wins − losses) |
|--------|---------------------|
| lt1 / EMG | **8** (8 побед, 0 поражений) |
| lt1 / EMG+NIRS | **8** |
| lt1 / EMG+NIRS+HRV | **8** |
| lt2 / EMG+NIRS+HRV | **8** |
| lt2 / EMG | 5 |
| lt2 / EMG+NIRS | 2 |
| lt1 / NIRS | 0 (ни побед, ни поражений) |
| lt2 / NIRS | 0 |

v0011 **не проигрывает ни одной NN-модели нигде**. На NIRS он не отличается значимо (net=0) — единственная модальность, где линейная модель не доминирует.

**Лидеры групп `noabs`** (v0011 там нет — нет noabs npy): единого доминанта среди NN нет, лидеры распределены: v0104 (lstm) ×3, v0106a (wavelet) ×3, v0106b ×2, v0101 ×1, v0107 ×1.

### Кросс-модальное сравнение чемпионов (v0011, with_abs)

| Сравнение | target | Δmedian | p | Вердикт |
|-----------|--------|---------|---|---------|
| HRV vs EMG+NIRS+HRV | lt2 | −0.10 | 0.966 | **n.s.** |
| HRV vs EMG+NIRS | lt2 | −0.71 | **0.012** | **HRV** значимо лучше |
| EMG+NIRS vs EMG+NIRS+HRV | lt1 | +0.10 | 0.442 | **n.s.** |

### Финальные победители

- **lt2**: v0011 на HRV. Причём `HRV ≈ EMG+NIRS+HRV` (p=0.97) — **мультимодальность не нужна, достаточно HRV**.
- **lt1**: v0011 на EMG+NIRS. Причём `EMG+NIRS ≈ EMG+NIRS+HRV` (p=0.44) — **добавление HRV не помогает, достаточно EMG+NIRS**.

### Где статистика не дала ответа

- На модальности **NIRS** v0011 не отличается значимо от NN (net=0 для обоих таргетов) — здесь «победителя» нет.
- В категории `noabs` нет сквозного доминанта — для каждой модальности свой лидер среди NN.

### Тезисы для диссертации

1. *«Регуляризованная линейная регрессия (v0011) статистически значимо превосходит все протестированные нейросетевые архитектуры (paired Wilcoxon с поправкой Холма, p < 0.05) на 6 из 8 конфигураций модальность × порог, не уступая нигде.»*
2. *«Для предсказания LT2 одной модальности ВСР достаточно: добавление ЭМГ и NIRS не даёт статистически значимого улучшения (p = 0.97). Для LT1 достаточно ЭМГ+NIRS: добавление ВСР не значимо (p = 0.44).»*
