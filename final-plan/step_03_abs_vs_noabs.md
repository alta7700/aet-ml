# Шаг 02. Ablation `with_abs` vs `noabs`

## Гипотеза
Модель, обученная **только на трендах** (без абсолютных значений HRV/SmO₂/EMG-уровней), будет **не хуже** модели с абсолютными признаками. Если подтвердится — это самостоятельный результат для текста (про обобщаемость на новых испытуемых, у которых индивидуальные «нормы» сдвинуты).

## Входы
- `results/summary_all_versions.csv` (агрегаты `with_abs` и `noabs` для каждой версии)
- `results/<version>/per_subject.csv` — **только для тех версий**, где `variant` присутствует в per_subject (NN-модели; у v0011 переменная вырождена для части fset).

## Что считаем
Для каждой пары `(version, target, feature_set)`:
- `Δmedian_MAE = median(MAE_noabs) − median(MAE_with_abs)` по субъектам;
- **paired Wilcoxon** между MAE per-subject (`alternative='two-sided'`), `n_paired`;
- эффект-сайз `r = Z / sqrt(N)`;
- решающий вердикт: «эквивалентно» (|Δ|<0.3 мин ∧ p>0.05), «лучше with_abs», «лучше noabs».

## Артефакты
- `scripts/final_compare_abs_vs_noabs.py`
- `results/final/06_abs_vs_noabs.csv`
- `results/final/plots/abs_vs_noabs_paired.png` — paired-line plot (with_abs → noabs) по субъектам для каждой пары `(version, fset, target)`
- Короткая подпись в `results/final/README.md` с выводом по гипотезе

## Ограничения
- Для v0011 EMG строки `with_abs` и `noabs` идентичны → исключаем.
- Используем только субъектов, у которых есть **обе** версии в per_subject.
- Holm-поправка не применяется — это исследовательский ablation, не подтверждающий.

## Критерий завершённости
- В CSV есть строка для каждой возможной пары.
- В README сформулирован однозначный вывод: гипотеза подтверждена / частично / отвергнута, со ссылкой на конкретные строки.

- **Дата**: 2026-05-14
- **Скрипт**: `scripts/final_abs_vs_noabs.py`
- **Артефакты**:
  - `results/final/11_abs_vs_noabs.csv` (72 строки = 9 NN-версий × 2 target × 4 fset)
  - `results/final/plots/abs_vs_noabs_volcano.png`

| Эквивалентно | **62/72 = 86%** |

### Вывод по гипотезе

**Гипотеза подтверждена и перевыполнена.** Модель, обученная только на относительной динамике (трендах), **не хуже** модели с абсолютными значениями признаков в 86% конфигураций, а в 7 случаях из 10 неэквивалентных — **значимо лучше**. Единственный случай в пользу абсолютных признаков — одна конфигурация v0102 с малым эффектом (0.20 мин).

### Что для текста диссертации

Самостоятельный результат:
> «Абсолютные базовые уровни физиологических сигналов (исходные значения HRV, SmO₂, EMG-RMS) не несут дополнительной информации для предсказания аэробно-анаэробного перехода. Модель, использующая только относительную динамику признаков, обобщается на новых испытуемых не хуже, а в ряде архитектур — значимо лучше (paired Wilcoxon, p < 0.05). Это объясняется высокой межиндивидуальной вариативностью абсолютных норм: в кросс-субъектной схеме они вносят шум, тогда как тренды универсальны.»

Практическое следствие: для внедрения **не требуется индивидуальная калибровка абсолютных норм** — достаточно отслеживать динамику. Это упрощает протокол.

## Уточнение: ablation для линейной модели v0011 (по запросу)

Шаг 03 выше считался **только по 9 NN-версиям** — v0011 был исключён (нет `noabs` npy). Поэтому вывод «абсолютные значения не важны» относится к нейросетям. Для линейной модели v0011 картина **другая** — проверено отдельно по `summary_all_versions.csv`.

### Что убирает `noabs` (EXCLUDE_ABS)
6 признаков: `trainred_smo2_mean`, `trainred_hhb_mean`, `trainred_hbdiff_mean`, `trainred_thb_mean`, `hrv_mean_rr_ms`, `feat_smo2_x_rr`. Для модальности HRV убирается **только один** — `hrv_mean_rr_ms` (среднее RR-интервала = абсолютный уровень ЧСС в окне). Вариабельностные признаки (`hrv_sdnn_ms`, `hrv_rmssd_ms`, `hrv_sd1/sd2`, `hrv_sd1_sd2_ratio`, `hrv_dfa_alpha1`) остаются и в noabs.

### v0011 / HRV / lt2 — ablation по summary

| вариант | raw_MAE | R² | ρ |
|---------|---------|-----|---|
| with_abs | 2.25 | **0.82** | 0.91 |
| noabs (без `hrv_mean_rr_ms`) | 2.84 | **0.68** | 0.85 |

Удаление **одного** признака `hrv_mean_rr_ms` роняет R² с 0.82 до 0.68. Для линейной модели абсолютный уровень ЧСС критичен — в отличие от NN.

### Коэффициенты линейной модели (мини-проверка)

Скрипт `scripts/final_v0011_hrv_coefs.py` обучил Ridge(α=1000) на HRV / lt2, стандартизованные коэффициенты (`results/final/11b_v0011_hrv_coefs.csv`):

| Признак | \|станд. коэф.\| | Доля |
|---------|------------------|------|
| `hrv_mean_rr_ms` (абс. уровень ЧСС) | 175.0 | **44.6%** |
| `hrv_dfa_alpha1` (DFA-α1, трендовый) | 108.3 | **27.6%** |
| `hrv_sd1_sd2_ratio` | 27.8 | 7.1% |
| `hrv_sd2_ms` | 23.3 | 6.0% |
| `hrv_sdnn_ms` | 22.0 | 5.6% |
| `hrv_rmssd_ms` | 17.8 | 4.5% |
| `hrv_sd1_ms` | 17.8 | 4.5% |

**Абсолютный якорь: 44.6%. Вариабельность/тренд: 55.3%.** Почти поровну.

### Вывод (подтверждена гипотеза «абсолют + тренд»)

Линейная модель v0011 на HRV опирается **одновременно** на:
- абсолютный якорь `hrv_mean_rr_ms` (44.6%) — универсальный кросс-субъектный маркер «где мы по шкале нагрузки» (абсолютная ЧСС монотонно растёт у всех);
- динамику/вариабельность (55.3%), где главный — `hrv_dfa_alpha1` (27.6%), известный физиологический маркер перехода.

Это **не «запоминание»**: модель валидирована LOSO, на отложенных субъектах работает. `hrv_mean_rr_ms` несёт кросс-субъектный сигнал, а не субъект-специфичную память. Если бы модель держалась только на абсолюте — noabs обвалился бы; он даёт R²=0.68 → трендовая компонента работает самостоятельно.

### Почему ablation разный для NN и linear

- **NN** видят весь временной ряд окон → могут реконструировать «положение на шкале нагрузки» из последовательности. Явный `hrv_mean_rr_ms` им избыточен → `with_abs ≈ noabs`.
- **Linear v0011** обрабатывает каждое окно как независимый вектор, без памяти о прошлых окнах → абсолютный признак — единственный способ «знать позицию на шкале». Убери его → потеря якоря, R² падает.

Это объясняет расхождение: для NN абсолютные признаки избыточны, для линейной модели — необходимы. Для текста диссертации: выбор «нужны ли абсолютные признаки» **зависит от класса модели**.
