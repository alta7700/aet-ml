## Слайд 12 — Результат 5: SHAP-интерпретация

**Главный контраст (заголовок-акцент):** концентрированный LT2 vs диффузный LT1.

**Обязательные числа:**

**LT2 / HRV (Ridge, LinearExplainer; табл. 3.13):**

| Признак | Доля mean \|SHAP\|, % |
|---|---|
| `hrv_mean_rr_ms` (абсолютный) | **43,5** |
| `hrv_dfa_alpha1` (трендовый) | **31,4** |
| `hrv_sd1_sd2_ratio` | 6,6 |
| остальные 4 признака ВСР | по 4–6 |

→ **75 % важности на двух признаках.**

**LT1 / EMG+NIRS (SVR + PermutationExplainer; табл. 3.15–3.16):**
- Ведущий признак `trainred_hhb_std` — **5,2 %**; топ-10 суммарно = 37,1 %.
- **Доминирующего признака нет — диффузный профиль.**
- По группам признаков: EMG 54,0 %, NIRS 40,3 %, кинематика 5,7 %.
- Для мульти-LT1 (EMG+NIRS+HRV): EMG 46,1 %, NIRS 31,2 %, HRV 11,6 %, признаки взаимодействия 6,8 %.

**Биологический смысл (две строки):**
- DFA-α1 = 0,5 — известный нелинейный маркер HRVT2 [Rogers, Gronwald 2022; Tanner 2024] → концентрация SHAP вокруг dfa_α1 + mean_rr физиологически предсказуема.
- Для LT1 концентрированного маркера в литературе нет; диффузный SHAP отражает физиологическую неоднородность LT1 (доставка O₂ у тренированных vs митохондрии у нетренированных).

**Визуальная подсказка:** **два SHAP-bar рядом**, один контрастен другому — это главный визуальный приём слайда. Слева — LT2/HRV (один высокий столбик 43,5 %, второй 31,4 %, остальные низкие). Справа — LT1/EMG+NIRS+HRV (плоский профиль, все столбики невысокие). Контраст должен быть очевиден с первого взгляда.

## Слайд 13 — Результат 6: надёжность (конформные интервалы)

**Цель слайда:** оценить практическую применимость моделей — насколько узкие интервалы можно гарантировать.

**Обязательная таблица (выборка из табл. 3.17, 80 % покрытие, with_abs):**

| Модель | Полуширина 80 %, мин | Эмпир. покрытие |
|---|---|---|
| Ridge / HRV (LT2) | **±3,49** | 0,800 |
| Ridge / EMG+NIRS+HRV (LT2) | ±4,18 | 0,803 |
| Ridge / EMG+NIRS (LT1) | ±6,99 | 0,828 |
| Ridge / EMG+NIRS+HRV (LT1) | ±6,33 | 0,837 |

**Ключевые тезисы:**
- Эмпирическое покрытие отклоняется от целевого на 0,2–2,2 п.п. для всех 16 строк — **гарантия LOSO split-conformal выполнена**.
- **LT2 — интервалы практически применимы:** ±3,5 мин при 80 % покрытии, что приемлемо для онлайн-обратной связи.
- **LT1 — пока слишком широки:** ±7 мин при 80 %, ±9 мин при 90 % — недостаточно для оперативного использования в текущей версии.
- Удаление абсолютных признаков расширяет интервалы LT2 (HRV ±3,49 → ±4,87 мин, +1,38 мин), но не меняет LT1 (даже сужает на 0,4 мин для EMG+NIRS).

**Практический вывод (одна строка-акцент):** LT2-система готова к практическому применению; LT1-система требует расширения выборки и поиска концентрированного маркера для сужения интервалов.

**Визуальная подсказка:** график конформных интервалов (полуширина по моделям) как основной элемент; таблица — компактный спутник. Цветом выделить контраст «практически применимо (LT2, зелёный) vs пока широко (LT1, жёлтый / красный)».
