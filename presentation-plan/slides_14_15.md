## Слайд 14 — Научная новизна

**Цель слайда:** сжато и формально перечислить, что нового внесла работа.

**Обязательный текст — 5 пунктов (§4.2), по одной строке на пункт:**

1. Количественно показана **зависимость минимально достаточного набора сенсоров от типа порога**: для LT2 одной ВСР достаточно (MAE 1,90 мин; добавление ЭМГ и NIRS не значимо, p = 0,966); для LT1 ведущей является комбинация ЭМГ+NIRS (MAE 2,11 мин; добавление ВСР не значимо, p = 0,442).

2. Установлено, что **выбор «абсолютные vs трендовые» признаки зависит от класса модели**: для нейросетевых архитектур — эквивалентность в 86 % конфигураций; для линейного семейства на LT2 / HRV абсолютные критичны (Δ = +0,54 мин, p = 0,006); ведущий абсолютный признак — `hrv_mean_rr_ms` (44,6 % коэффициента Ridge).

3. Получено эмпирическое подтверждение, что **при n = 18 регуляризованная линейная регрессия на ручных признаках статистически значимо превосходит все 9 нейросетевых архитектур** (90 значимых побед, 0 поражений в 14 из 16 групп; Уилкоксон + Холм).

4. Описан и количественно охарактеризован **эффект подгруппы тренированных для предсказания LT1**: per-subject MAE значимо выше на тренированных (p = 0,0002…0,003); 79 % окон с большой ошибкой (|err| > 7 мин) приходятся на эту подгруппу. Эффект отсутствует для LT2 (p = 0,35…0,68).

5. Показано, что **для LT1 при n = 18 симметричное разделение модели на двух специалистов не работает, тогда как асимметричный вариант** (исключение нетипичной подгруппы из обучающей выборки модели большинства) даёт значимый выигрыш при идеальной маршрутизации (p = 0,034) и тренд при реальной (p = 0,074); MAE 3,15 → 2,18 мин.

**Визуальная подсказка:** пять одинаково оформленных пунктов с лаконичными значками-маркерами. Каждый пункт визуально начинается с короткого слова-якоря (например, «модальности», «признаки», «архитектура», «подгруппа», «маршрутизация») — это упрощает запоминание.

## Слайд 15 — Ограничения и направления

**Цель слайда:** академически прозрачно обозначить ограничения и наметить следующий шаг.

**Обязательный текст — два блока:**

**Ограничения (§4.4, сжато):**
- n = 18 испытуемых; на отдельных нейросетевых конфигурациях LOSO даёт n = 16–17.
- Выборка гетерогенная — отклонилась от исходно целевой популяции триатлонистов; нельзя прямо сопоставлять с опубликованными нормами лактатного порога для профессионалов.
- Подгруппа тренированных n = 6 → выводы по системе двух экспертов **эксплоративны**.
- Маршрутизатор по 11 базальным признакам — точность LOSO 66,7 %, что совпадает с базовой частотой большего класса.
- LT1 conformal-интервалы ±7–9 мин — пока недостаточны для оперативного использования.

**Направления дальнейших исследований (§4.5):**
- Расширение выборки тренированных для проверки симметричной двухэкспертной схемы.
- Поиск концентрированного физиологического маркера LT1 — агрегированного признака, синтезирующего вклад множества слабых сигналов.
- Стратифицированная конформная калибровка по подгруппам тренированности.
- Расширение SHAP-интерпретации на нейросетевые архитектуры (DeepExplainer / Integrated Gradients).

**Визуальная подсказка:** двухколоночная вёрстка — слева «Ограничения» (нейтральный / жёлтый), справа «Направления» (зелёный / акцент). Подсветить, что каждое направление логически вытекает из конкретного ограничения слева (можно протянуть тонкие стрелки или пронумеровать соответствия).
