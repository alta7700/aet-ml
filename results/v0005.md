## Что делает

Два дополнительных анализа поверх моделей v0004:

1. **SHAP (SHapley Additive exPlanations)** — объяснение важности признаков. Для линейной модели SHAP-значение признака j для наблюдения i:
   ```
   φ_j(i) = coef_j × (x_j(i) − E[x_j])
   ```
   Глобальная модель обучается на всех данных (не LOSO). Визуализация: bar plot, beeswarm, траектории по участникам.

2. **Cross-conformal prediction** — доверительные интервалы с гарантией покрытия. Для каждого тестового субъекта s:
   - Для каждого тренировочного s': обучить модель на {все} \ {s, s'}, предсказать s' → nonconformity score = |y_true − ŷ|
   - Набрать n-1 калибровочных scores → квантиль Q(α)
   - Интервал предсказания: [ŷ(s) − Q, ŷ(s) + Q]

## Физиологическая интерпретация SHAP (LT2)

### hrv_mean_rr_ms (1.253 мин, rank #1)

Самый крупный вклад — в среднем 1.25 мин ошибки только от этого признака. Физиологически объяснимо: RR кодирует и динамику нагрузки внутри сессии (ρ_within = +0.997), и уровень тренированности (ρ_between = +0.753). Это не значит, что признак «плохой» — он несёт реальный сигнал. Но часть его вклада — межсубъектный конфаунд, а не физиологический механизм порога.

**Визуализация beeswarm** (если построена): ожидается чёткий красно-синий разрыв по оси x — высокий RR (красный) → положительный SHAP (ещё далеко до порога), низкий RR (синий) → отрицательный SHAP (близко к порогу).

### trainred_thb_mean (0.847 мин, rank #2)

SHAP-вклад ниже, чем у RR, но второй по величине. SHAP-значения неоднородны между участниками из-за разнонаправленного тренда THb (см. `common.md`). У участников с монотонным ростом THb — большой положительный SHAP в начале теста; у S011/S012 (обратный тренд) — знак может инвертироваться.

**Флаг потенциального неправильного фокуса:** если SHAP beeswarm показывает blobby pattern без чёткой красно-синей границы — признак работает как «шумовой якорь» для конкретных субъектов, а не как физиологический маркер.

### trainred_hhb_std (0.740 мин, rank #3) и trainred_thb_std (0.665 мин, rank #5)

std-признаки занимают три позиции в топ-5. Два конкурирующих объяснения:

1. **Дыхательные колебания NIRS**: частота дыхания нарастает с нагрузкой → амплитуда дыхательных осцилляций в NIRS растёт. При высоком std ещё «далеко» от порога в фазе роста нагрузки.
2. **Кинематические артефакты**: педалирование создаёт механические вибрации, видимые в NIRS. Если каданс меняется с нагрузкой, std может отражать это косвенно.

Отсутствие устойчивой корреляции с `cadence_mean_rpm` не опровергает гипотезу 2 — кадан может не меняться (постоянный темп педалирования), но гармонические составляющие в NIRS могут изменяться по амплитуде.

**Для диссертации:** зафиксировать как «потенциально информативный, механизм не установлен» — честная позиция при малой выборке.

### trainred_hbdiff_std (0.493 мин, rank #6) и trainred_hbdiff_mean (0.362 мин, rank #8)

HbDiff = HbO₂ − HHb — оксигенационный баланс. При приближении к LT2 HbO₂ падает быстрее, чем растёт HHb → hbdiff снижается. std hbdiff отражает нестабильность этого баланса.

Физиологически более осмысленный признак, чем отдельные HHb/HHb — разность компенсирует часть артефактов движения (одинаково влияющих на оба канала NIRS).

### feat_rr_per_watt (0.322 мин, rank #9) и trainred_smo2_mean (0.282 мин, rank #10)

**feat_rr_per_watt = hrv_mean_rr_ms / watt**: нормированный на мощность RR. Частично компенсирует влияние ступени нагрузки, но наследует конфаунд тренированности из RR.

**trainred_smo2_mean**: базовый NIRS-сигнал. Меньший SHAP по сравнению со std-признаками может указывать, что **уровень** SmO₂ менее информативен, чем **вариабельность** SmO₂ — это неочевидный, но физиологически возможный результат (важна нестабильность оксигенации, а не абсолютное значение).

## Выводы

- Conformal prediction успешно откалиброван при n=9–13: покрытие ≈ целевому (±1%)
- При α=0.10 ширина интервала ~9 мин — приемлемо для 30-минутного теста
- SHAP подтверждает доминирование hrv_mean_rr_ms (1.25 мин вклад на образец)
- NIRS признаки (thb_mean, hhb_std, hbdiff_std) занимают 2–6 позиции для LT2
- Для LT1 интерпретация SHAP затруднена мультиколлинеарностью HRV — ограничение метода
- Conformal intervals можно использовать в реальном протоколе: «ещё N мин, ±4.7 мин с 90% вероятностью»
