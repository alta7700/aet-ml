# v0007 — 6 стратегий вариабельности × зоопарк моделей

**Скрипт:** scripts/v0007_train_variability_features.py  
**Дата:** 2026-05-07  
**Предыдущая версия:** v0004  
**Данные:** results/v0007/  

## Что делает

Тестирует 6 стратегий добавления признаков вариабельности педалирования к базе v0004.
Каждая стратегия использует **один масштаб окна** — разные масштабы в одну модель не смешиваются.
Для каждой стратегии прогоняется полный **зоопарк из 6 моделей**: Ridge α=100, Huber ε=1.35, ElasticNet, SVR rbf, RF n=200, GBM n=200.

### Стратегии

| # | Стратегия | Признаки |
|---|---|---|
| 1 | Trend CV 30s | load/rest_trend_cv_slope_30s, load/rest_trend_cv_ratio_30s |
| 2 | Trend CV 60s | то же, 60с окно |
| 3 | Trend CV 120s | то же, 120с окно |
| 4 | SampEn 30s | load/rest_sampen_30s |
| 5 | SampEn 60s | load/rest_sampen_60s |
| 6 | SampEn 120s | load/rest_sampen_120s |

### Физиологическая гипотеза

Приближение к порогу → нарастание ритмической нестабильности педалирования.  
**Trend CV** ловит динамику нарастания вариабельности (CV в позднем третьем окна > CV в раннем).  
**SampEn** ловит потерю сложности ритма (нелинейная мера непредсказуемости каденса).

## Выводы

### Поведение модельного зоопарка

- **SVR rbf** — катастрофически плохо для обеих задач (3.0–3.3 мин). Нелинейность не помогает при данном объёме данных.
- **RF/GBM для LT1** — переобучаются даже на 10 участниках LOSO: RF≈2.7, GBM≈2.6 против Huber=2.1.
- **GBM для LT2** — с 14 участниками GBM больше не обходит Ridge. С ростом выборки дисперсия оценки падает, и GBM (2.094–2.120) уступает Ridge (2.023) на всём диапазоне конфигураций.
- **Линейные модели** (Ridge/Huber/ElasticNet) — стабильны и надёжны для обеих задач.

### Влияние признаков вариабельности

- **Trend CV для LT2** — лучший вариант по числам (Trend CV 60s + Ridge = 2.023), но улучшение Δ=-0.000 находится в пределах шума LOSO. Статистически незначимо.
- **SampEn для LT2** — не помогает линейным моделям; GBM с SampEn 30s = 2.108, что хуже базового Ridge = 2.023. С 14 участниками эффект исчезает.
- **SampEn 30s для LT1** — единственный устойчивый эффект: Huber c SampEn 30s = 2.110 против 2.118 базы (Δ=-0.008). SampEn 120s ухудшает результат — при длинном окне меньше циклов вблизи порога, оценка нестабильнее.
- **Физиология SampEn:** оба коэффициента (load_sampen_30s и rest_sampen_30s) отрицательные — более высокая непредсказуемость ритма → ближе к LT1. Соответствует нейромышечной дестабилизации у аэробного порога.

### Итог

Признаки вариабельности не дают значимого улучшения для LT2 (Δ≈0). Для LT1 SampEn 30s даёт Δ=-0.008 мин. Комбинации признаков — v0008.
