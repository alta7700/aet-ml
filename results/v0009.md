# v0009 — Kalman-постобработка поверх лучших конфигураций v0008

**Скрипт:** scripts/v0009_kalman_postprocess.py  
**Дата:** 2026-05-07  
**Предыдущая версия:** v0008  
**Данные:** results/v0009/  

## Что делает

Применяет 1D Kalman-сглаживатель как постобработку к LOSO-предсказаниям лучших конфигураций из v0008.

**Физическое обоснование:** истинное время до порога τ(t) убывает примерно на шаг окна за каждый шаг. Модель выдаёт шумные снэпшоты τ̂_t. Kalman объединяет предыдущее состояние (prior) с новым наблюдением:

```
Модель состояния:  τ_{t+1} = τ_t − step + w,  w ~ N(0, σ_p²)
Наблюдение:        z_t = τ_t + v,              v ~ N(0, σ_obs²)
```

Параметры выбираются grid-поиском на LOSO-данных:
- σ_p ∈ {15, 30, 60} с
- σ_obs ∈ {60, 90, 120, 150} с

**Конфигурации:**
- LT2: v0008★ SampEn 30s + GBM n=200 (raw MAE=2.043)
- LT1: v0008★ SampEn 60s+timing_cv + Huber ε=1.35 (raw MAE=1.959)
- + v0004-ref для контроля

## Гиперпараметры

| Параметр | Значение |
|---|---|
| Шаг окна (step) | реальный из window_start_sec (≈30с) |
| Инициализация | τ_0 = первое предсказание, P_0 = σ_obs² |
| Лучшие σ_p / σ_obs | 15с / 150с (оба таргета) |

## Результаты

| σ_p \ σ_obs | 60с | 90с | 120с | 150с |

| 30с | 2.010 | 1.998 | 1.988 | 1.979 |
| 60с | 2.027 | 2.018 | 2.010 | 2.003 |

★ Лучшая: σ_p=15с, σ_obs=150с → **MAE=1.943**, Δ=-0.100 vs raw

★ Лучшая: σ_p=15с, σ_obs=150с → **MAE=1.901**, Δ=-0.058 vs raw

## Выводы

- Kalman даёт стабильный Δ≈-0.06÷-0.10 мин независимо от задачи и базовой модели.
- Лучшие параметры одинаковы для обоих таргетов (σ_p=15с, σ_obs=150с), что говорит о физической обоснованности: σ_p=15с отражает умеренную плавность убывания τ, σ_obs=150с (~2.5 мин) соответствует порядку LOSO MAE.
- Kalman-gain при этих параметрах: K ≈ 10–15%, т.е. фильтр реально интегрирует новые наблюдения, а не просто экстраполирует — честное сглаживание.
- Граница поиска σ_obs достигнута (150с — крайнее значение сетки). Расширение сетки → v0010.
