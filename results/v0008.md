# v0008 — Комбинации признаков вариабельности × зоопарк моделей

## Что делает

Тестирует комбинации лучших признаков вариабельности из v0007 (SampEn 30s, SampEn 60s, timing_cv) поверх базы v0004. Для каждой из 8 комбинаций признаков — полный **зоопарк из 6 моделей**.

### Комбинации признаков

| Конфигурация | Доп. признаки | Кол-во |
|---|---|---|
| v0004-репликация | — | 32/14 |
| SampEn 30s | load/rest_sampen_30s | +2 |
| SampEn 60s | load/rest_sampen_60s | +2 |
| timing_cv | load/rest_duration_cv | +2 |
| SampEn 30s+60s | sampen 30s + sampen 60s | +4 |
| SampEn 30s+timing_cv | sampen 30s + timing_cv | +4 |
| SampEn 60s+timing_cv | sampen 60s + timing_cv | +4 |
| SampEn 30s+60s+timing_cv | sampen 30s + sampen 60s + timing_cv | +6 |

## Гиперпараметры

Идентичны v0007 (те же 6 моделей в зоопарке, те же Imputer и Scaler).

## Выводы

### LT2

Для LT2 **GBM — единственная модель, которой помогают новые признаки**:
- SampEn 30s + GBM = 2.043 (лучшее)
- Комбинирование SampEn с timing_cv GBM не улучшает (2.074+)
- Линейным моделям новые признаки на LT2 не помогают или мешают

Вывод: **оптимум LT2 — SampEn 30s + GBM n=200**.

### LT1

Для LT1 картина богаче — все линейные модели выигрывают от новых признаков:
- **timing_cv** (Δ=-0.041 на Huber) — самый информативный одиночный добавок
- **SampEn + timing_cv** (Δ=-0.062) — лучшая комбинация, стабильна на Ridge и ElasticNet
- **SampEn 30s+60s** без timing_cv — слабее (Δ=-0.008), добавление второго окна SampEn не помогает
- Три признаковые группы вместе (30s+60s+timing_cv) — чуть хуже двух (Δ=-0.054)

Вывод: **оптимум LT1 — SampEn 60s+timing_cv + Huber ε=1.35** (или SampEn 30s+timing_cv — идентичный MAE).

### Физиологическая интерпретация новых признаков

- **load_duration_cv → раньше** (-46.4): рост вариабельности длительности рабочей фазы = нейромышечная нестабилизация у LT1
- **load_sampen_60s → раньше** (-19.7): более непредсказуемый ритм в 60с окне → ближе к LT1
- **rest_duration_cv → позже** (+15.7): рост вариабельности паузы — возможно адаптивное удлинение отдыха при высокой нагрузке
