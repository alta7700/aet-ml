## Где что повторяется

| Сущность | Где фигурирует | Требование |
|---|---|---|
| Тема | Приказ → Титул → ТЗ | Дословное совпадение |
| Цель | ТЗ → Реферат → Abstract → Введение → Заключение | Одна формулировка |
| Задачи | ТЗ → Введение → Результаты → Заключение | Каждая закрыта результатом и выводом |
| Методы | Реферат → Введение → Обзор → М/М → Результаты | В реферате — перечень; в М/М — детали; в Результатах — применение |
| Результаты | Реферат (сжато) → Гл. 3 (подробно) → Заключение (резюме) | Одна интерпретация |
| Новизна | Реферат → Введение → Заключение | Вытекает из gap'а из Обзора |
| Объём (стр/рис/табл) | Первая строка Реферата | Считать после финальной вёрстки |
| Ключевые слова | Реферат → Abstract | 5–15, ПРОПИСНЫЕ, именительный падеж |
| Сокращения | Раздел «Сокращения» ↔ текст | Двустороннее соответствие |
| Источники | Ссылки в тексте ↔ Список | Нумерация по порядку первого упоминания |
| Таблицы/рисунки | Гл. 2–3, Приложения | Ссылка в тексте до объекта; нумерация в пределах раздела |

## Чеклист перед сдачей

- [ ] Тема совпадает дословно: приказ = титул = ТЗ
- [ ] Цель одинакова во всех пяти местах
- [ ] Каждая задача закрыта результатом и выводом; выводов ≥ задач
- [ ] Каждый метод из реферата описан в М/М и применён в Результатах
- [ ] Нет источника без ссылки; нет ссылки без источника
- [ ] Нумерация источников — по первому упоминанию
- [ ] Двустороннее соответствие аббревиатур
- [ ] На каждую таблицу/рисунок — ссылка до появления
- [ ] На каждое приложение — ссылка в тексте; все в оглавлении
- [ ] Числа в реферате соответствуют финальной вёрстке
