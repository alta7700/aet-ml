# Порядок сборки диссертации

**Главное:** порядок написания почти обратен порядку в итоговом PDF.

## Последовательность

| № | Что пишется | Место в PDF | Примечание |
|---|---|---|---|
| 1 | ТЗ | 2-й элемент | Цель и задачи — контракт; не менять после утверждения |
| 2 | Материалы и методы | Гл. 2 | Пока детали свежи |
| 3 | Результаты (черновик) | Гл. 3 | Чистые факты, без обсуждения |
| 4 | Обзор литературы | Гл. 1 | После результатов — тогда ясно, что цитировать |
| 5 | Обсуждение | Гл. 3 | Сравнение с обзором, интерпретация, ограничения |
| 6 | Введение | Перед главами | Структура: актуальность → оценка состояния → цель (из ТЗ) → задачи (из ТЗ) → методология → теоретическая значимость → практическая значимость |
| 7 | Заключение | После гл. | Вывод по каждой задаче ТЗ; выводов ≥ задач |
| 8 | Список источников | После заключения | По порядку первого упоминания (п. 4.10.4) |
| 9 | Сокращения | 5-й элемент | Если аббревиатур > 3 |
| 10 | Рефераты рус/англ | 3–4-й элементы | Последними — нужны финальные числа; Abstract — не перевод, а параллельный текст |
| 11 | Вёрстка + пагинация | — | Поля, шрифт, нумерация объектов |
| 12 | Оглавление | 6-й элемент | После финальной пагинации |
| 13 | Титульный лист | 1-й элемент | Тема дословно совпадает с ТЗ и приказом |
| 14 | Приложения (проверка) | Последние | Обозначения А, Б, В, … — без Ё, З, Й, О, Ч, Ь, Ы, Ъ; ссылка в тексте обязательна; нумерация страниц сквозная |

## Структура реферата

```
[XX страниц, XX рисунков, XX таблиц, XX источников, XX приложений]
[КЛЮЧЕВЫЕ СЛОВА]
Объект. Цель. Методы. Результаты и новизна. Практическая значимость.
```
≤ 850 знаков без пробелов.
