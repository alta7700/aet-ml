# План литературного обзора

## Структура

```
1. Физиология аэробно-анаэробного перехода
2. ЭМГ как маркер порога при велоэргометрии
   2.1 Классические признаки: RMS, MDF
   2.2 Вейвлетный анализ ЭМГ
3. NIRS и мышечная оксигенация
4. HRV и DFA-α1
   4.1 DFA-α1 = 0.75 → VT1
   4.2 DFA-α1 = 0.50 → VT2
5. Кинетика лактата
6. ML для детекции порога
   6.1 Классический ML на ручных признаках
   6.2 ML для временных рядов (LSTM, TCN)
   6.3 Малые датасеты и валидация
   6.4 Gap: мультимодальные онлайн-системы
7. Физиология популяции и протоколы тестирования
8. Заключение по обзору (формулировка gap → переход к цели)
```

---

## Содержание разделов

**1. Физиология перехода** — биохимия LT1/LT2/MLSS, рекрутирование МЕ, почему два порога.

**2.1 ЭМГ классика** — нелинейный рост RMS, сдвиг MDF, breakpoint на кривой EMG–мощность, разделение по фазам педалирования через гироскоп.

**2.2 Вейвлеты** — DWT-энергия по уровням db4 как признак, вейвлет-энтропия; преимущество перед FFT для нестационарного сигнала. *Gap: применение при ступенчатом велотесте для детекции порога не систематизировано.*

**3. NIRS** — SmO₂ как локальный маркер, breakpoint оксигенации при VT1/VT2, ограничения (жир, артефакты движения). *Gap: мало работ с одновременным ЭМГ + NIRS на одной мышце.*

**4.1 DFA-α1 = 0.75** — теоретическое обоснование, валидность у тренированных (Gronwald, Rogers, Schaffarczyk), ограничения (длина окна, артефакты RR).

**4.2 DFA-α1 = 0.50** — потеря фрактальной структуры при VT2; Rogers 2022, Tanner 2024. Менее валидировано, чем 0.75 для VT1.

**5. Кинетика лактата** — задержка капиллярного vs мышечного лактата, влияние длительности ступени, преимущество велоэргометрии (меньше межмышечного шаттлинга).

**6.1 Классический ML** — RF/SVM/XGBoost для классификации зон; changepoint (CUSUM, ruptures) для разметки; SHAP для интерпретируемости.

**6.2 Временные ряды** — LSTM/TCN vs ручные признаки; Zignoli et al. 2019 (RNN для VT). При n < 50 ручные признаки + классический ML выигрывают.

**6.3 Малые датасеты** — LOSO как обязательная валидация; почему k-fold недопустима; аугментация (jittering, time warping).

**6.4 Gap (центральный)** — нет систем, объединяющих ЭМГ + NIRS + HRV для предсказания VT1 и VT2 одновременно в реальном времени. Нет валидации на разнородной выборке.

**7. Популяция и протоколы** — диапазоны мощностей LT1/LT2, стандартизация ступенчатых протоколов.
