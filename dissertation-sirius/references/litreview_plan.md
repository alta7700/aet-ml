## Детализированный план с содержанием каждого подраздела

### 1. Физиология аэробно-анаэробного перехода
- Биохимия лактатного порога, различие VT1/LT1 и VT2/LT2/MLSS
- Рекрутирование двигательных единиц при нарастании нагрузки
- Почему детектируются два порога, а не один
- *Gap: нет надёжного неинвазивного онлайн-метода детекции обоих порогов одновременно*

---

### 2. ЭМГ как маркер порога при велоэргометрии

#### 2.1 Классические признаки: RMS, MDF
- Нелинейный рост RMS при приближении к порогу (breakpoint на кривой EMG–мощность)
- Сдвиг медианной частоты MDF вниз при мышечной усталости
- Паттерны активации VL и RF по фазам педалирования (push/pull) через IMU/гироскоп
- Ограничения: межэлектродные артефакты, необходимость нормализации

#### 2.2 Вейвлетный анализ ЭМГ ← НОВЫЙ ПОДРАЗДЕЛ
- Почему вейвлеты лучше FFT для нестационарного ЭМГ-сигнала: одновременное разрешение по времени и частоте
- DWT (дискретное): энергия по уровням декомпозиции как признаки для классификации
  - Уровни db4 декомпозиции для ЭМГ 2000 Гц: d1 (~500–1000 Гц) — шум, d3–d5 (~50–250 Гц) — основная активность
  - Агрегаты: энергия `sum(d²)`, дисперсия, вейвлет-энтропия (WE)
  - Преимущество перед RMS: раздельная мощность в каждом частотном диапазоне, чувствительность к рекрутированию МЕ
- CWT (непрерывное): скалограмма для визуализации частотно-временной динамики при смене ступени
- Wavelet CNN: вейвлетные фильтры как обучаемый первый слой нейросети
  - Разница от DWT: параметры фильтров обучаются под задачу, не фиксированы
  - Применимость: требует больше данных, чем ручные признаки + классический ML
- Рекомендуемые вейвлеты для ЭМГ: Daubechies db4 (наиболее распространён), Morlet (для CWT)
- *Gap: большинство вейвлет-работ — изолированная усталость или статические сокращения; применение при ступенчатом велотесте для детекции порога не систематизировано*

---

### 3. NIRS и мышечная оксигенация
- SmO2 как маркер локального кислородного баланса в мышце
- Breakpoint оксигенации и его соответствие VT1 и VT2
- Преимущества перед системными маркерами: непрерывность, локальность
- Ограничения: артефакты движения, подкожный жир, калибровка
- *Gap: мало работ с одновременным ЭМГ + NIRS на одной мышце*

---

### 4. HRV и DFA-α1 как онлайн-маркер VT1 и VT2

#### 4.1 DFA-α1 = 0.75 → HRVT1 (VT1)
- Вегетативные сдвигипри нарастании нагрузки, RMSSD
- DFA-α1: теоретическое обоснование, значение α1 = 0.75 как маркер VT1
- Валидность у тренированных спортсменов (Gronwald, Rogers, Schaffarczyk)
- Ограничения: длина окна, артефакты RR, необходимость стационарности

#### 4.2 DFA-α1 = 0.50 → HRVT2 (VT2) ← НОВЫЙ ПОДРАЗДЕЛ
- При интенсивности VT2 HRV теряет фрактальную структуру: α1 снижается до ~0.50
  (переход от персистентного к случайному поведению)
- Теоретическое обоснование: полная потеря вегетативного контроля ЧСС при
  анаэробной нагрузке
- Ключевые источники: Rogers 2022 ("HRV Fractal Correlation for Endurance
  Training Prescription"), Tanner 2024 ("Agreement Between HRV-Derived and
  Ventilatory Lactate Thresholds") — оба есть в lith
- **Ограничение:** α1 = 0.50 для VT2 менее валидировано, чем 0.75 для VT1;
  большинство исследований фокусировались на VT1; необходима явная ссылка на
  конкретные работы при использовании в диссертации
- *Gap: DFA-α1 = 0.50 как маркер VT2 требует дополнительной верификации,
  особенно на триатлонистах*

---

### 5. Кинетика лактата
- Задержка между мышечным и капиллярным лактатом (временная и концентрационная)
- Влияние длительности ступени на точность определения порога
- Влияние активной мышечной массы: велоэргометрия (изолированная нагрузка на ноги) vs лыжи/гребля (межмышечный шаттлинг лактата)
- Методологическое преимущество велоэргометрии: меньше шаттлинга → системный лактат точнее отражает локальный метаболизм VL/RF
- NIRS на VL как независимый локальный сигнал, не зависящий от задержки системного лактата

---

### 6. Машинное обучение для детекции порога

#### 6.1 Классический ML на ручных признаках
- Обзор существующих подходов: RF, SVM, XGBoost для классификации зон интенсивности
- Входные признаки: RMS, MDF, DFA-α1, SmO2, вейвлетная энергия (из раздела 2.2)
- Методы changepoint detection: CUSUM, BOCPD, ruptures — для офлайн-разметки и онлайн-детекции
- Преимущества при малых данных: меньше параметров, лучше обобщение, интерпретируемость через SHAP

#### 6.2 ML для временных рядов ← НОВЫЙ ПОДРАЗДЕЛ
- LSTM/GRU: память о предыдущих состояниях, естественны для физиологических сигналов с временной зависимостью
- TCN (Temporal Convolutional Network): параллельное обучение, управляемое рецептивное поле, быстрый инференс
- Wavelet CNN в контексте ML (см. также раздел 2.2): автоматическое извлечение частотных признаков
- Ключевая работа: Zignoli et al. (2019) — RNN для классификации вентиляционных порогов
- Сравнение: ручные признаки + классический ML vs сырой сигнал + LSTM/TCN
  - Ручные признаки выигрывают при n < 50 субъектов (меньше риск переобучения)
  - LSTM/TCN выигрывают при n > 200 и длинных непрерывных записях

#### 6.3 Малые датасеты и валидация ← НОВЫЙ ПОДРАЗДЕЛ
- Почему n = 30–40 — типичный размер в спортивной физиологии и что это означает для ML
- Leave-One-Subject-Out (LOSO) как обязательный метод валидации при малых данных
  - Почему обычная k-fold недопустима: утечка данных между субъектами
- Методы борьбы с переобучением: регуляризация, ограничение числа признаков, ансамблевые методы
- Аугментация временных рядов: jittering, time warping, window slicing — применимость к ЭМГ и HRV
- Transfer learning для физиологических сигналов: предобучение на большом датасете → дообучение на малом
- Вывод: при n = 30–40 классический ML на ручных признаках + LOSO — наиболее обоснованный выбор

#### 6.4 Gap: мультимодальные онлайн-системы
- Существующие работы используют каналы по отдельности: только ЭМГ, только NIRS, только HRV
- Нет систем, объединяющих ЭМГ + NIRS + HRV для предсказания **двух** порогов (VT1 и VT2) **одновременно** в **реальном времени**
- Нет работ с вероятностным выходом P(LT1) и P(LT2) в скользящем окне
- Нет валидации мультимодального подхода именно на триатлонистах
- Это обоснование новизны данной диссертации

---

### 7. Физиология триатлонистов и протоколы тестирования
- Типичные значения VO2max, мощности на LT1/LT2 у любителей и профессионалов
- Стандартизация ступенчатых протоколов для тренированных спортсменов
- Почему триатлонисты — подходящая популяция: высокая аэробная база, мотивация, воспроизводимость
