# Карта диссертации: что в каком разделе

## Поток информации

```
ТЗ (цель + задачи)
  → Введение (актуальность + те же цель/задачи + gap в одной фразе)
    → Обзор (контекст, gap явно, основания для методов)
      → М/М (как именно)
        → Результаты (закрывает каждую задачу)
          → Заключение (вывод по задаче → новизна → значимость)
            → Реферат (всё в 850 знаков)
```

Оглавление и список источников — после финальной вёрстки.

---

## Что новое появляется в каждом разделе

| Раздел | Новые сущности |
|---|---|
| ТЗ | Тема, цель, задачи — контракт |
| Реферат | Объём в числах (стр/рис/табл/ист/прил), ключевые слова |
| Сокращения | Список аббревиатур, если их > 3; в оглавление не входит |
| Введение | Актуальность, оценка состояния проблемы, теоретическая значимость |
| Обзор (Гл. 1) | Авторы, методы предшественников, явный gap |
| М/М (Гл. 2) | Детали методов, биообъекты с идентификаторами, статистика |
| Результаты (Гл. 3) | Данные, таблицы/рисунки со ссылками до появления, сравнение с обзором |
| Заключение | Вывод по каждой задаче, научная новизна, ограничения, перспективы |
| Приложения | Только если есть ссылка в тексте; нумерация сквозная |

---

## Чеклист согласованности

- [ ] Тема совпадает дословно: приказ = титул = ТЗ
- [ ] Цель одинакова во всех пяти местах: ТЗ / реферат / abstract / введение / заключение
- [ ] Каждая задача закрыта результатом и выводом; выводов ≥ задач
- [ ] Каждый метод из реферата описан в м/м и применён в результатах
- [ ] Нет источника без ссылки в тексте и ссылки без источника в списке
- [ ] Нумерация источников — по первому упоминанию (п. 4.10.4)
- [ ] Каждая аббревиатура в разделе «Сокращения» встречается в тексте, и наоборот
- [ ] На каждую таблицу и рисунок — ссылка до его появления
- [ ] На каждое приложение — ссылка в тексте; все приложения в оглавлении
- [ ] Числа в реферате соответствуют финальной вёрстке
- [ ] Gap в обзоре явно обосновывает новизну из заключения
