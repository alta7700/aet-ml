## Опорные факты (передавать каждому агенту)

- Выборка: n = 18 испытуемых, протокол LOSO-кросс-валидации.
- Задача: регрессия `time_to_LT1_sec`, `time_to_LT2_sec`.
- Модальности признаков: EMG, NIRS, HRV, EMG+NIRS, EMG+NIRS+HRV.
- Три варианта MAE:
  - subject-weighted median = **1,90 мин** (LT1, EMG+NIRS+HRV);
  - window median = 3,33 мин;
  - raw window-weighted = 4,25 мин.
- Победители inner-моделей (по комбинации модальность × порог):
  - LT2 / HRV → Ridge (α = 1000)
  - LT2 / EMG+NIRS+HRV → ElasticNet (α = 1,0; l1_ratio = 0,9)
  - LT1 / EMG+NIRS → **SVR (C = 100, ε = 1,0; RBF-ядро)**
  - LT1 / EMG+NIRS+HRV → **SVR (C = 100, ε = 1,0; RBF-ядро)**
  - LT1 / EMG → GBM (n_estimators = 100, max_depth = 2)
- Финальный ансамбль (внутренний код `v0107`) = Wavelet-TCN + ElasticNet.
- Статистика: парный критерий Уилкоксона с поправкой Холма, U-критерий Манна—Уитни; конформные интервалы (LOSO split-conformal).
- Двухэкспертная гипотеза подтверждена в асимметричном варианте: p = 0,034 (оракул-роутер) и p = 0,074 (реальный роутер).
